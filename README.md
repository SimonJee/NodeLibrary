NodeLibrary
===========

High level Arduino library for wireless communication using the JeeNode (jeelabs.org) hardware. 

Requirements
===========

* Arduino >= 1.0 
* JeeLib (http://github.com/jcw/jeelib)


Documentation & Source
===========

Currently maintained at github.com
